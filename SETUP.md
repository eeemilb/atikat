# Быстрая настройка ATIKAT

## Шаг 1: Подготовка Supabase

1. Откройте https://supabase.com/dashboard
2. Выберите ваш проект
3. Перейдите в **Storage** → нажмите **Create bucket**
4. Название: `topics` (или своё)
5. Отметьте это **Public bucket**
6. Нажмите **Create**

## Шаг 2: Исправьте код

В этом репозитории найдите файл `index.html`

Найдите строку:

```javascript
const STORAGE_BUCKET = 'topics';
```

Если вы дали bucket другое имя — исправьте значение.

## Шаг 3: Добавьте файлы в Storage

### На этом этапе вы:
1. Вернитесь в Supabase Dashboard
2. Откройте bucket `topics`
3. Создавайте папки и загружайте `.md` и `.txt` файлы

### Пример структуры:

```
топикы/
├── электроника/
│   ├── circuits.md
│   └── microcontrollers.md
├── проекты/
│   ├── edm/
│   │   └── description.md
│   └── pcb.md
└── докс/
    └── intro.md
```

## Шаг 4: Проверьте

1. Откройте последнюю версию приложения (https://atikat.vercel.app)
2. Нажмите на вкладку "Файлы"
3. Она должна с автоматически загружены с ваши топики

## От настных в производство

Приложение автоматически синхронизируется с Supabase Storage. Пыстать бы обновить топикы в любом моменте:

1. Только откройте приложение заново
2. Нажмите на "Файлы"
3. Всё всё синхронизируется автоматически!

## Поддерживаются форматы

- **Markdown** (`.md`) — лучший выбор
- **Текст** (`.txt`)
- **Другие** — отображаются сырым текстом

## Вы сделали!

Теперь всё готово. Добавляйте новые топики в Supabase Storage — и сервис автоматически их навендет к доступным!
